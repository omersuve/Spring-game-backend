## Technologies Used
* Spring
* MySQL
* Redis
* Unittest

## How to run

-  JDK version 21
-  Maven

```bash
mvn clean install
```

```bash
java -jar -DDB_USERNAME=<your_mysql_db_username> -DDB_PASSWORD=<your_my_sql_db_password> target/dream-backend-0.0.1-SNAPSHOT.jar
```
